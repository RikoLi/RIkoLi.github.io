GraphSAGE：Node Embedding初探&复现的一些想法

一开口，其实有些惭愧，不是因为摸鱼太久没有干活，而是本科生必须要经历的破事情太多（写不得不完成的专业课作业，当初脑子一热报的SRTP…），隔了很久才将实验室的活正式开工，读过的paper稍微有点生疏。

废话不多说了，这篇文章总结一下Stanford一篇NIPS paper，是实验室一位博士生安排给我尝试复现的一个GNN延伸的模型，主要做node embedding，即图神经网络点嵌入。

点嵌入：将新的节点加入已存在的图中，确定其边的连接关系。

前言

文章首先总结了传统的利用GNN进行embedding的手段，大致有如下几点特征：

  • 使用整张图$G(V, E)$的信息，每个节点的特征都拿去学习
  • 图是固定的，节点之间的关系不可改变
  • 难以泛化至未出现过的新图

换句话说，传统方法使用一张已经确定了所有节点关系的图进行训练，然后使用训练好的模型对新的节点进行嵌入。

本文提出GraphSAGE方法，通过取样，每次使用不同size的minibatch进行训练，分深度选择邻居节点。设置一个depth-limitation，训练过程中更新所有节点的参数（用向量表示，包括新嵌入点与原图中节点）。

算法

分三个部分：

  • 前向传播
  • 反向传播
  • 前向传播中minibatch size改变策略
